2. install 方式

	2.1. 到 C: 建立目錄 SysJustMCPServer, 產生 c:\SysJustMCPServer
	
	2.2. 複製
		\\j-nas01\部門-研發一部\ServerTeam\Component
		MCP_Main_XQ_TWStock.py
		tools
		到 c:\SysJustMCPServer 完成後該目錄下應有
			MCP_Main_XQ_TWStock.py
			tools

	2.3. 執行 python-3.14.3-amd64.exe 安裝python 3.14，安裝目錄C:\Python314
	
	2.4. 以上是因為要對應claude_desktop_config.json到MCP Server 安裝路徑

		"XQ-TWStock": {
		  "command": "C:\\Python314\\python.exe",
		  "args": [
			"C:\\SysJustMCPServer\\MCP_Main_XQ_TWStock.py"
		  ]


	2.5. 另外因為有用到 python 以下的 package
		from fastmcp import FastMCP
		from typing import Annotated, List
		from pydantic import BaseModel, Field
		import requests
		import xml.etree.ElementTree as ET

		故到 C:\Python314\Scripts
		dos commadn 執行方式
		cd C:\Python314\Scripts>
		pip install fastmcp 
		pip install requests
		
		其它package應該已經有了，若有問題可下以下的 package
		pip install typing
		pip install pydantic
		pip install xml
